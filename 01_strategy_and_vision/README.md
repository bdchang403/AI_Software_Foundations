# Strategy Lead Hub: The Source of Context

As a Strategy Lead, you are the "Source" of the Context Factory. Your strategy sets the boundaries for everyone else. If your input is fuzzy, the AI's output will be hallucinatory.

## 🤝 The Context Handshake

| Inbound | Role Action | Outbound |
| :--- | :--- | :--- |
| **Business Strategy** | **Context Bounding** | **Semantic Blueprint** |
| (Generic Goals) | (Defining Constraints) | (AI-Ready Strategy) |
