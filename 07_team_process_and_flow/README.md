# Welcome Agile Leader: Monitoring the Factory

The Context Factory represents a fundamental shift in software delivery. AI agents can write hundreds of lines of code in seconds, but if the context fed to them is flawed, they will confidently write *the wrong code* at scale. 

As an Process Coach, Engineering Manager, or Scrum Master, you are the **System Optimizer**. You do not write the 3-Part Directives; your job is to observe the telemetry emitted by the BuilderOps pipelines to identify where the "Context Hand-Off" is failing.
